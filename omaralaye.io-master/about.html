About the Pig Disease Detection System

Overview

This system combines IoT and AI to monitor pig health, helping farmers detect illnesses early and improve livestock management.

Components

  • ESP32-CAM: Captures images or video for analysis.
  • Machine Learning Model: Detects health status using Edge Impulse or server-based models.
  • Flutter App: Displays live feed, health status, and alerts.

How It Works

  1. The ESP32-CAM captures images when triggered (e.g., motion detection).
  2. Images are processed to classify pig health (healthy or sick).
  3. Results are sent to the Flutter app via Wi-Fi or MQTT.

Benefits

  • Early detection of health issues to reduce mortality.
  • Cost-effective monitoring with affordable hardware.
  • User-friendly app interface for farmers.

System Workflow

ESP32-CAM → Machine Learning Model → Flutter App
